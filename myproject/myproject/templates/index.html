Welcome to LMN architecture!

Introducing the decentralized version of architecture.

Initial statement

This system was made for the competition algodeq. To show a new method of designing, using collective intelligence. The goal for the participating users will be simply modeling the best model, evaluated by the other users in the system. However, there is one major limitation, that the users will be forced to inherit another model which already exists.

tell me more about it.

このシステムはalgodeqというコンペのために、開発されたものです。分散バージョン管理という方法を使った新しいデザインの可能性を追求します。 システムに参加するユーザーの目的は、「最も評価されるモデル」を作ることですが、「すでにあるモデルを継承したほうが有利に働きやすい」特殊なルールがあります。

Behind the screens.

This web application uses webGL. the model creation process will be done one your current modern browser. The system also uses the diff algorithm to calculate the ratio of the differences within models.

tell me more about it.
